# 0. 개요

이 튜토리얼은 하나의 플러그인을 만들어 보면서 Kotlin 으로 Bukkit 플러그인을 만드는 방법을 배우게 될 것입니다.

각 튜토리얼마다 사용된 코드와, 개발 환경은 GitHub 에 같이 올려두어 언제든지 확인 가능하도록 할 예정입니다.

추가로 궁금한 사항이 있다면, GitHub 의 Discussion 기능을 적극 활용해주시면 감사하겠습니다!
